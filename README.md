# Agent coursework

Reinforcement learning and multi-agent coursework for the 2026 HICHEEL work tree.

## Contents

- Lab notebooks and Python scripts for DQN, PPO, A2C, and multi-agent communication experiments.
- `Archive/`: earlier training, inference, and evaluation scripts.
- `biydaalt/`: assignment-specific MAGent/MAPPO-style experiments.
- `agentcrew/`: CrewAI project scaffold and notes.

## Artifact policy

Model weights, checkpoints, generated videos, GIFs, logs, and archives are intentionally excluded from Git. Put those files in a Hugging Face repository or dataset instead of GitHub.

See `HF_ARTIFACTS.txt` for the local artifact list that was present when this repository was prepared.
